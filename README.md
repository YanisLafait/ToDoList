# ToDoList

- Etape 1 : Créer un formulaire HTML pour permettre à l'utilisateur de saisir les tâches à ajouter à la liste.
- Etape 2 : Demander les tâches pour la To Do List.
- Etape 3 : Ajouter une fonctionnalité pour supprimer les tâches de manière automatique
- Etape 4 : Utiliser CSS pour créer un design responsive
- Etape 5 : Ajouter une fonctionnalité pour gérer les dates. Vérifier si la date de la tâche est passée et supprimer la tâche automatiquement si c'est le cas.
